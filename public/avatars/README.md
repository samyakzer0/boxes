# Avatar Images

This directory contains avatar images for the Marquee component.

## Required Images

Please add 6 avatar images named:
- jack.png
- jill.png
- john.png
- jane.png
- jenny.png
- james.png

Recommended specs:
- Format: PNG or WebP
- Size: 40x40 pixels (or 80x80 for 2x displays)
- Optimized for web

## Placeholder Generation

You can generate these using:
1. **UI Avatars** - https://ui-avatars.com/api/?name=Jack&size=40
2. **Boring Avatars** - https://boringavatars.com
3. **DiceBear** - https://www.dicebear.com/
4. Or create custom avatars in your design tool

## Security Note

Using local assets instead of external CDNs:
- ✅ Eliminates external dependencies
- ✅ No third-party tracking
- ✅ Better performance (no DNS lookup)
- ✅ Works offline
